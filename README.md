# 망가번역기 [VRAM 최소 12기가 이상(권장 16기가 이상)의 NVIDIA RTX GPU만 지원합니다!!!!]

망가번역기는 일본어 만화 페이지를 가져와 한국어 번역 결과를 만들고, 화면에서 바로 손볼 수 있게 해주는 Windows 데스크톱 앱입니다.

이미지 몇 장만 열어서 가볍게 작업할 수도 있고, 폴더나 ZIP 파일로 여러 화를 한 번에 가져와 보관함에 정리해둘 수도 있습니다. 번역이 끝난 뒤에는 박스를 옮기고, 크기를 바꾸고, 문장을 직접 수정하면서 결과를 다듬을 수 있습니다.

## 할 수 있는 일

- `이미지 열기`, `폴더 열기`, `압축파일 열기`로 페이지를 보관함에 추가
- `작품 일괄 번역`으로 ZIP 여러 개를 한 번에 가져와 여러 화 처리
- 새 작품으로 저장하거나 기존 작품에 이어 붙이기
- 작품/화 검색, 이름 변경, 순서 정리
- `이어서 번역`, `전체 다시 번역`, 페이지별 `재번역`
- 번역된 박스 이동, 크기 조절, 텍스트 수정
- 페이지, 화, 작품 삭제
- 설정에서 번역 엔진, 번역 모드, 모델 프리셋, `GPU layers`, `NSFW 모드` 변경
- 로그 폴더 바로 열기

## 사용 방법

1. 앱에서 `이미지 열기`, `폴더 열기`, `압축파일 열기`, `작품 일괄 번역` 중 하나를 선택합니다.
2. 새 작품을 만들지, 기존 작품에 추가할지 선택합니다.
3. 화 제목을 확인한 뒤 보관함에 추가합니다.
4. 번역이 끝나면 페이지를 열어 필요한 부분만 직접 수정합니다.
5. 덜 끝난 화는 `이어서 번역`, 처음부터 다시 하고 싶으면 `전체 다시 번역`을 사용합니다.

`작품 일괄 번역`은 선택한 폴더 안의 ZIP 파일들과, 하위 폴더 안에 이미지가 들어 있는 폴더들을 여러 화로 읽어 순서대로 처리합니다.

## 저장 위치

- 개발 실행에서는 보관함이 `library/`에, 로그가 `logs/app.log`에 저장됩니다.
- 설치형 배포본에서는 실행 파일 옆 `data/` 폴더를 사용합니다.
- 보관함은 `data/library`
- 로그는 `data/logs`
- 설정은 `data/settings.json`
- 모델 캐시는 `data/hf-cache`

## 설정

앱은 기본으로 Gemma 4 기반 설정을 사용합니다. 설정 화면에서 OpenAI Codex/openai-oauth 기반 번역 엔진도 선택할 수 있고, 변경 내용은 다음 번 번역부터 적용됩니다.

- 번역 엔진 `Gemma 4`, `OpenAI Codex`
- 번역 모드 `빠름`, `정확성`
- 모델 프리셋 `Q4`, `Q6`, `커스텀`
- Codex 모델 기본값 `gpt-5.5`, 생각 `없음`, `낮음`, `보통`, `높음`, `최고`
- `GPU layers` 값
- `NSFW 모드`

OpenAI Codex 엔진은 `openai-oauth`가 Codex 로그인 정보를 사용합니다. 처음 사용할 때는 먼저 `codex login` 또는 `npx @openai/codex login`으로 로그인해두세요.
