# 망가번역기

일본어 만화 이미지를 불러와 한국어로 번역하고, 번역된 문장을 화면에서 직접 고칠 수 있는 Windows 데스크톱 앱입니다.

이미지 몇 장만 열어서 가볍게 작업할 수도 있고, 폴더나 ZIP 파일로 여러 화를 한 번에 가져와 보관함에 정리할 수도 있습니다. 번역이 끝난 뒤에는 말풍선 박스를 옮기고, 크기를 바꾸고, 문장을 직접 수정하면서 결과를 다듬을 수 있습니다.

## 먼저 알아둘 점

이 앱에는 번역 엔진이 2가지 있습니다.

- `OpenAI Codex`: OpenAI 계정으로 로그인해서 번역합니다. 내 컴퓨터에 고성능 그래픽카드가 없어도 사용할 수 있습니다.
- `Gemma 4`: 내 컴퓨터에서 직접 AI 모델을 돌려 번역합니다. 이 방식만 고성능 NVIDIA RTX 그래픽카드가 필요합니다.

그래서 예전 README 제목에 있던 `VRAM 최소 12기가 이상(권장 16기가 이상)의 NVIDIA RTX GPU만 지원` 문구는 앱 전체가 아니라 `Gemma 4` 엔진에만 해당합니다. `OpenAI Codex` 엔진을 쓰면 RTX GPU 요구사항이 없습니다.

## 어떤 엔진을 고르면 되나요?

처음 쓰는 분에게는 `OpenAI Codex` 엔진을 먼저 권장합니다.

- 컴퓨터 사양을 덜 탑니다.
- 모델 파일을 직접 받을 필요가 없습니다.
- 기본 모델은 `gpt-5.5`로 설정되어 있습니다.
- 설정에서 생각 수준을 `없음`, `낮음`, `보통`, `높음`, `최고` 중에서 고를 수 있습니다.

`Gemma 4` 엔진은 인터넷 없이 로컬에서 돌리고 싶거나, OpenAI 계정을 쓰지 않고 내 PC에서만 처리하고 싶을 때 선택하면 됩니다. 대신 아래 조건을 만족해야 합니다.

- NVIDIA RTX GPU 필요
- VRAM 최소 12GB 이상
- VRAM 16GB 이상 권장
- AMD/Intel GPU, 내장 그래픽, VRAM이 부족한 그래픽카드는 지원하지 않습니다.

## 처음 사용하는 방법

1. 앱을 실행합니다.
2. 오른쪽 위 설정을 엽니다.
3. `번역 엔진`에서 원하는 엔진을 고릅니다.
4. `OpenAI Codex`를 고른 경우, 먼저 터미널에서 `codex login` 또는 `npx @openai/codex login`으로 OpenAI 로그인을 해둡니다.
5. 앱에서 `이미지 열기`, `폴더 열기`, `압축파일 열기`, `작품 일괄 번역` 중 하나를 선택합니다.
6. 새 작품을 만들지, 기존 작품에 추가할지 선택합니다.
7. 화 제목을 확인한 뒤 보관함에 추가합니다.
8. 번역이 끝나면 페이지를 열어 필요한 부분만 직접 수정합니다.

터미널이 익숙하지 않다면 Windows 시작 메뉴에서 `PowerShell`을 검색해 실행한 뒤, 위 로그인 명령어를 그대로 붙여 넣으면 됩니다.

## 할 수 있는 일

- `이미지 열기`, `폴더 열기`, `압축파일 열기`로 페이지를 보관함에 추가
- `작품 일괄 번역`으로 ZIP 여러 개를 한 번에 가져와 여러 화 처리
- 새 작품으로 저장하거나 기존 작품에 이어 붙이기
- 작품/화 검색, 이름 변경, 순서 정리
- `이어서 번역`, `전체 다시 번역`, 페이지별 `재번역`
- 번역된 박스 이동, 크기 조절, 텍스트 수정
- 페이지, 화, 작품 삭제
- 설정에서 번역 엔진, 번역 모드, 모델 프리셋, `GPU layers`, `NSFW 모드` 변경
- 로그 폴더 바로 열기

## 작업 흐름

1. 만화 이미지, 이미지 폴더, ZIP 파일 중 하나를 앱에 넣습니다.
2. 앱이 페이지를 보관함에 저장합니다.
3. 선택한 번역 엔진으로 일본어 문장을 한국어로 번역합니다.
4. 결과 화면에서 번역 박스를 직접 확인합니다.
5. 어색한 문장은 바로 수정합니다.
6. 덜 끝난 화는 `이어서 번역`, 처음부터 다시 하고 싶으면 `전체 다시 번역`을 사용합니다.

`작품 일괄 번역`은 선택한 폴더 안의 ZIP 파일들과, 하위 폴더 안에 이미지가 들어 있는 폴더들을 여러 화로 읽어 순서대로 처리합니다.

## 설정 설명

앱은 기본으로 `Gemma 4` 기반 설정을 사용합니다. 설정 화면에서 `OpenAI Codex` 기반 번역 엔진으로 바꿀 수 있고, 변경 내용은 다음 번 번역부터 적용됩니다.

- 번역 엔진: `Gemma 4`, `OpenAI Codex`
- 번역 모드: `빠름`, `정확성`
- 모델 프리셋: `Q4`, `Q6`, `커스텀`
- Codex 모델 기본값: `gpt-5.5`
- Codex 생각 수준: `없음`, `낮음`, `보통`, `높음`, `최고`
- `GPU layers`: Gemma 4에서 GPU를 얼마나 사용할지 정하는 값
- `NSFW 모드`: 성인향 이미지 번역을 허용하는 설정

`OpenAI Codex` 엔진은 `openai-oauth`가 Codex 로그인 정보를 사용합니다. 처음 사용할 때는 먼저 `codex login` 또는 `npx @openai/codex login`으로 로그인해두세요.

## 저장 위치

- 개발 실행에서는 보관함이 `library/`에, 로그가 `logs/app.log`에 저장됩니다.
- 설치형 배포본에서는 실행 파일 옆 `data/` 폴더를 사용합니다.
- 보관함은 `data/library`
- 로그는 `data/logs`
- 설정은 `data/settings.json`
- 모델 캐시는 `data/hf-cache`

## 문제가 생겼을 때

번역이 안 되면 먼저 설정에서 어떤 엔진을 선택했는지 확인하세요.

- `OpenAI Codex`를 쓰는 경우: OpenAI 로그인이 되어 있는지 확인합니다.
- `Gemma 4`를 쓰는 경우: NVIDIA RTX GPU와 충분한 VRAM이 있는지 확인합니다.
- 앱이 멈춘 것처럼 보이는 경우: 큰 ZIP 파일이나 고해상도 이미지는 시간이 오래 걸릴 수 있습니다.
- 오류 내용을 확인하고 싶은 경우: 앱의 로그 폴더 열기 기능을 사용합니다.
